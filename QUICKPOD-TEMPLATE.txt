## Advanced Configuration

### Docker Options (copy everything below this line):
```
--gpus all \
-p 6080:6080 \
-p 5900:5900 \
-p 22:22 \
-p 8080:8080 \
-e SSH_USERNAME=reaper \
-e SSH_PASSWORD=changeme123 \
-e VNC_PASSWORD=changeme123 \
-e PUID=1000 \
-e PGID=1000 \
-e TZ=Etc/UTC \
-e RESOLUTION=1920x1080x24 \
-e DISPLAY=:1 \
-e NVIDIA_VISIBLE_DEVICES=all \
-e NVIDIA_DRIVER_CAPABILITIES=all \
-v /workspace:/home/reaper/projects \
-v /plugins/vst2:/home/reaper/.wine/drive_c/Program\ Files/Common\ Files/VST2 \
-v /plugins/vst3:/home/reaper/.wine/drive_c/Program\ Files/Common\ Files/VST3 \
-v /config:/home/reaper/.config \
--shm-size=2gb \
--device /dev/snd \
--group-add audio
```

### On Start Scripts:
```
#!/bin/bash
echo "Setting up WineReaper environment..."

# Create necessary directories
mkdir -p /home/reaper/projects \
         /home/reaper/.wine/drive_c/Program\ Files/Common\ Files/VST2 \
         /home/reaper/.wine/drive_c/Program\ Files/Common\ Files/VST3 \
         /home/reaper/.config/REAPER

# Set up audio permissions
usermod -a -G audio reaper

# Initialize Wine prefix (first time only)
if [ ! -f /home/reaper/.wine/.initialized ]; then
    echo "Initializing Wine prefix..."
    wine wineboot --init
    winetricks -q corefonts vcrun2019 dotnet48
    touch /home/reaper/.wine/.initialized
fi

echo "WineReaper ready! Access via:"
echo "- Web GUI: http://[your-instance]:6080/vnc.html"
echo "- VNC Client: [your-instance]:5900 (password: changeme123)"
echo "- SSH: ssh reaper@[your-instance] (password: changeme123)"
```

## Usage Instructions

### After deployment:
1. **Web Access**: http://[your-instance]:6080/vnc.html
   - Password: `changeme123`
2. **VNC Client**: Connect to [your-instance]:5900
   - Password: `changeme123`
3. **SSH Access**: `ssh reaper@[your-instance]`
   - Password: `changeme123`

### Volume Mounts:
- `/home/reaper/projects`: Reaper project files and audio
- `/plugins/vst2`: VST2 plugins (persistent)
- `/plugins/vst3`: VST3 plugins (persistent)
- `/config`: Reaper configuration and preferences

### Included Software:
- Reaper DAW (Windows version in Wine)
- Pre-installed free VSTs (TAL, Dexed, Surge XT, Vital)
- GPU-accelerated plugin support
- VNC/noVNC web interface
- Full audio/MIDI support

## Notes
- **GPU Required**: For plugin acceleration and real-time processing
- **Audio Setup**: May need host audio device passthrough (`--device /dev/snd`)
- **First Run**: Wine initialization may take 5-10 minutes
- **Plugin Installation**: Copy VST files to mounted plugin directories
- **Performance**: Use `--shm-size=2gb` for better plugin performance
- **Storage**: 100GB recommended for sample libraries and projects
